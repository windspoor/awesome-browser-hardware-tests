# Contributing

Thanks for your interest in contributing to Awesome Browser Hardware Tests.

You can suggest a new tool by opening an issue or pull request.

A good submission should be:

- Browser-based
- Useful for testing hardware, devices, peripherals, or performance
- Free to use or have a useful free version
- Easy to access without complicated setup
- Relevant to gamers, creators, hardware users, or troubleshooting

Please include:

- Tool name
- URL
- Short description
- Suggested category

Spammy, misleading, or unrelated submissions may be rejected.

# Awesome Browser Hardware Tests
[![Awesome](https://awesome.re/badge.svg)](https://awesome.re)
A curated list of free browser-based tools for testing hardware, gaming devices, and peripherals.

Most tools in this list work directly in the browser and do not require downloads, drivers, or software installation.

This list is useful for gamers, creators, remote workers, students, repair shops, hardware reviewers, and anyone who wants to quickly check whether a device is working properly.

## Community

Join the discussion on [r/HardwareTest](https://www.reddit.com/r/HardwareTest/) for tool suggestions, feedback, and browser-based hardware testing discussions.

## Featured Tools

- [HardwareTest.org](https://hardwaretest.org/) - All-in-one browser-based hardware testing tools.
- [Mouse Polling Rate Test](https://mousepollingratetest.com/) - Test mouse polling rate, peak Hz, jitter, and stability.
- [Controller Test Online](https://controllertestonline.com/) - Test game controller buttons, joysticks, triggers, and stick drift.

## All-in-One Hardware Testing

* [HardwareTest.org](https://hardwaretest.org/) - A collection of free browser-based tools for testing monitors, keyboards, mice, audio devices, webcams, controllers, click speed, and more.

## Mouse Testing

* [Mouse Polling Rate Test](https://mousepollingratetest.com/) - Test your mouse polling rate, average Hz, peak Hz, jitter, and stability directly in the browser.
* [TestUFO Mouse Poll Rate Test](https://testufo.com/mouserate) - Browser-based mouse polling rate and jitter test.
* [DeviceTests Mouse Test](https://devicetests.com/mouse-test) - Test mouse buttons, scroll wheel, and basic mouse input.
* [CPS Test Polling Rate Test](https://cpstest.org/polling-rate-test/) - Online mouse polling rate checker.
* [Click Speed Tester Mouse Rate Checker](https://clickspeedtester.com/mouse-polling-rate-checker/) - Check average and maximum mouse polling rate online.

## Keyboard Testing

* [Keyboard Tester](https://www.keyboardtester.com/) - Test whether keyboard keys are working correctly.
* [DeviceTests Keyboard Test](https://devicetests.com/keyboard-test) - Browser-based keyboard key test.
* [Keyboard Test Online](https://keyboard-test.space/) - Online keyboard testing tool for checking key input.
* [Key-Test](https://en.key-test.ru/) - Simple online keyboard checker.

## Game Controller Testing

* [Controller Test Online](https://controllertestonline.com/) - Test game controller buttons, joysticks, triggers, D-pad input, connection status, and possible stick drift directly in the browser.
* [Gamepad Tester](https://hardwaretester.com/gamepad) - Test gamepads, buttons, axes, and controller input online.
* [DeviceTests Gamepad Tester](https://devicetests.com/controller-test) - Browser-based controller and gamepad testing tool.
* [Online Gamepad Tester](https://gamepadtester.net/) - Test gamepad buttons, joysticks, and controller input.
* [Gamepad Viewer](https://gamepadviewer.com/) - Visualize gamepad input for testing, streaming, and controller display.

## Monitor and Screen Testing

* [Dead Pixel Test](https://deadpixeltest.org/) - Test your monitor for dead pixels, stuck pixels, and display issues.
* [EIZO Monitor Test](https://www.eizo.be/monitor-test/) - Professional monitor test patterns for checking display quality.
* [Lagom LCD Monitor Test Pages](http://www.lagom.nl/lcd-test/) - Classic LCD monitor test pages for contrast, sharpness, gamma, and response.
* [DeviceTests Dead Pixel Test](https://devicetests.com/dead-pixel-test) - Browser-based dead pixel and screen test.
* [Blur Busters UFO Test](https://www.testufo.com/) - Motion, refresh rate, frame skipping, and display performance tests.

## Webcam Testing

* [Webcam Test](https://webcamtests.com/) - Test webcam image, resolution, frame rate, and device status.
* [DeviceTests Webcam Test](https://devicetests.com/webcam-test) - Simple online webcam test.
* [OnlineMicTest Webcam Test](https://www.onlinemictest.com/webcam-test/) - Test whether your webcam is working in the browser.

## Microphone and Audio Testing

* [Online Mic Test](https://www.onlinemictest.com/) - Test microphone input directly in the browser.
* [DeviceTests Microphone Test](https://devicetests.com/microphone-test) - Browser-based microphone test.
* [MicTests.com](https://mictests.com/) - Check microphone input, playback, and device status.
* [Left Right Stereo Test](https://www.audiocheck.net/audiotests_stereo.php) - Test left and right speaker channels.

## Internet and Network Testing

* [Speedtest by Ookla](https://www.speedtest.net/) - Test internet speed, ping, and connection quality.
* [Fast.com](https://fast.com/) - Simple internet speed test by Netflix.
* [Cloudflare Speed Test](https://speed.cloudflare.com/) - Test download, upload, latency, jitter, and packet loss.
* [Packet Loss Test](https://packetlosstest.com/) - Test packet loss, latency, and connection stability.

## Benchmark and Performance Testing

* [Basemark Web](https://web.basemark.com/) - Browser-based performance benchmark.
* [BrowserBench Speedometer](https://browserbench.org/Speedometer3.0/) - Measure web app responsiveness and browser performance.
* [MotionMark](https://browserbench.org/MotionMark1.3/) - Browser graphics performance benchmark.
* [WebGL Aquarium](https://webglsamples.org/aquarium/aquarium.html) - WebGL graphics performance demo and stress test.

## Contributing

Contributions are welcome.

To suggest a new tool, please open an issue or pull request.

A good tool for this list should be:

* Browser-based
* Useful for testing hardware, devices, peripherals, or performance
* Free to use or have a useful free version
* Accessible without complicated setup
* Relevant to gamers, creators, hardware users, or general device troubleshooting

Please avoid submitting spammy, misleading, or unrelated tools.
